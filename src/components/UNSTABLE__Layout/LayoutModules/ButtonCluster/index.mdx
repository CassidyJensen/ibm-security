# ButtonCluster

## Overview

Button clusters group secondary actions that can be taken on a whole page or component. Icons are used to help describe, at a glance, the action that can be taken.

You can build a button cluster using a combination of the `ButtonCluster` and [`Button`](#references) components. The `ButtonCluster` accepts a list of `Button` components as `children`.

## Grid placement

When placing a button cluster on the grid, it should be placed in 4 columns.

```jsx
<Column lg={4}>
  <ButtonCluster>
    <Button kind="ghost">Button 1</Button>
    <Button kind="ghost">Button 2</Button>
  </ButtonCluster>
</Column>
```
