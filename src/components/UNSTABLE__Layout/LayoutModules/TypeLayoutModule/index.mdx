# TypeLayoutModule

## Overview

Type layout modules provide a means to orderly layout terms and definitions.

You can build a type layout module using a combination of the `TypeLayoutModule`, [`TypeLayout`, and `Title`](#references) components.

The `TypeLayoutModule` accepts `TypeLayout`, and optionally `Title`, components as `children`.

## Grid placement

When placing a type layout module on the [grid](#references), it should be placed in a maximum of 8 columns.

```jsx
<Column lg={8}>
  <TypeLayoutModule>
    <TypeLayout>
      <TypeLayoutBody>
        <TypeLayoutRow>
          <TypeLayoutCell>Term 1</TypeLayoutCell>
          <TypeLayoutCell>Definition 1</TypeLayoutCell>
        </TypeLayoutRow>

        <TypeLayoutRow>
          <TypeLayoutCell>Term 2</TypeLayoutCell>
          <TypeLayoutCell>Definition 2</TypeLayoutCell>
        </TypeLayoutRow>

        <TypeLayoutRow>
          <TypeLayoutCell>Term 3</TypeLayoutCell>
          <TypeLayoutCell>Definition 3</TypeLayoutCell>
        </TypeLayoutRow>
      </TypeLayoutBody>
    </TypeLayout>
  </TypeLayoutModule>
</Column>
```
