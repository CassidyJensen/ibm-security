# TypeLayout

## Overview

Type layout provide a means to orderly layout terms and definitions.

You can build a type layout using a combination of the `TypeLayout` and [`Title`](#references) components.

The `TypeLayout` accepts `TypeLayoutBody`, `TypeLayoutCell`, and `TypeLayoutRow` components as `children`.

If necessary, both section and subsection titles can precede the type layout.

## Grid placement

When placing a type layout module on the [grid](#references), it should be placed in a maximum of 8 columns.

```jsx
<Column lg={8}>
  <TypeLayout>
    <TypeLayoutBody>
      <TypeLayoutRow>
        <TypeLayoutCell>Term 1</TypeLayoutCell>
        <TypeLayoutCell>Definition 1</TypeLayoutCell>
      </TypeLayoutRow>

      <TypeLayoutRow>
        <TypeLayoutCell>Term 2</TypeLayoutCell>
        <TypeLayoutCell>Definition 2</TypeLayoutCell>
      </TypeLayoutRow>

      <TypeLayoutRow>
        <TypeLayoutCell>Term 3</TypeLayoutCell>
        <TypeLayoutCell>Definition 3</TypeLayoutCell>
      </TypeLayoutRow>
    </TypeLayoutBody>
  </TypeLayout>
</Column>
```
