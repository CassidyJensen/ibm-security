# `ButtonClusterModule`

## Overview

Button cluster modules group secondary actions that can be taken on a whole page or component. Icons are used to help describe, at a glance, the action that can be taken.

You can build a button cluster module using a combination of the `ButtonClusterModule` and <Link kind="components-button--default">`Button`</Link> components. The `ButtonClusterModule` accepts a list of `Button` components as `children`.
