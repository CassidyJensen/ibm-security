## Overview

The page tab module can be built using a combination of the `PageTab`, [`Tabs`](#references), and `PageTabDetails` components.

The `PageTab` accepts `Tabs`, and optionally `PageTabDetails`, components as `children` and each `Tab` is responsible for displaying the tab's label and panel content.

## Grid placement

Following the [narrow grid](#references) implementation, page tab modules should be placed directly into the grid context.

```jsx
<Grid>
  <PageTab>
    <Tabs>
      <Tab label="Tab 1">
        <Row>
          <Column>Tab 1 content</Column>
        </Row>
      </Tab>

      <Tab label="Tab 2" />
      <Tab label="Tab 3" />
    </Tabs>
  </PageTab>
</Grid>
```
