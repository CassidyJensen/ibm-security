## Overview

The ICA module can be built using a combination of the `ICAModule`, [grid](#references), and [`ICA`](#references) components.

Everything passed in as a [render prop](#references) child of `ICAModule` will be rendered.

Additional props passed into `ICAModule` will be forwarded along to the underlying ICA module container.

## Grid placement

ICAs within ICA modules can span across 2 or 3 columns, following the narrow grid implementation.

<Row>
  <Column>
    <img alt="#" src="#" />
  </Column>

  <Column>
    <img alt="#" src="#" />
  </Column>
</Row>

```jsx
<ICAModule>
  {() => (
    <Row>
      <Column lg={3}>
        <ICA />
      </Column>

      <Column lg={3}>
        <ICA />
      </Column>

      <Column lg={3}>
        <ICA />
      </Column>
    </Row>
  )}
</ICAModule>
```
