## Overview

The ICA module can be built using a combination of the `ICAModule`, [grid](#references), [`ICA`](#references), and optionally the `withHover`, [higher-order component](#references).

Everything passed in as a child of `ICAModule` will be rendered.

## withHover

The `withHover` higher-order component enhances columns to provide consistent hover styling when using an ICA within the ICA module.

```jsx
const ColumnWithHover = withHover(Column);

return (
  <ICAModule>
    <Row>
      <ColumnWithHover>
        <ICA />
      </ColumnWithHover>

      <ColumnWithHover>
        <ICA />
      </ColumnWithHover>

      <ColumnWithHover>
        <ICA />
      </ColumnWithHover>
    </Row>
  </ICAModule>
);
```

## Grid placement

ICAs within ICA modules can span across 2 or 3 columns, following the narrow grid implementation.

  <Column>

```jsx
<ICAModule>
  <Row>
    <Column lg={3}>
      <ICA />
    </Column>

    <Column lg={3}>
      <ICA />
    </Column>

    <Column lg={3}>
      <ICA />
    </Column>
  </Row>
</ICAModule>
```
