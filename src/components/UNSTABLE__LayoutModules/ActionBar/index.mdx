## Overview

The action bar module can be built using a combination of the `ActionBar` and `ActionBarItems` components.

`ActionBarItems` is responsible for positioning the actions, and everything passed in as a child of `ActionBarItems` will be rendered.

## Grid placement

Action bar modules can span between 4 and 16 columns, following the [narrow grid](#references) implementation.

```jsx
<Column lg={4}>
  <ActionBar>
    <Button kind="ghost">Action 1</Button>

    <ActionBarItems>
      <IconButtonBar
        actions={[
          {
            label: 'Action 2',
            renderIcon: Add16,
          },
          {
            label: 'Action 3',
            renderIcon: Edit16,
          },
          {
            label: 'Action 4',
            renderIcon: Filter16,
          },
        ]}
        size="md"
      />
    </ActionBarItems>
  </ActionBar>
</Column>
```
